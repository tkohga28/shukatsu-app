## 1. アプリの概要

| 項目 | 内容 |
|---|---|
| アプリ名 | 就跡（しゅうせき） |
| URL | https://shuseki.vercel.app |
| サブタイトル | 就活の軌跡を積み上げよう |
| ターゲット | 27卒・28卒の就活生 |
| 料金 | 無料 |
| プラットフォーム | Webアプリ（PWA対応・スマホのホーム画面に追加可能） |

---

## 2. アプリの機能一覧

| 機能 | 内容 |
|---|---|
| 面接回答フィードバック | 質問と回答を入力するとAIが結論・根拠・学び・再現性の4軸で採点・FB |
| ESフィードバック | 企業名・設問・回答を入力するとAIが面接での深掘り耐性を観点にFB |
| 志望動機チェック | 企業名と志望動機を入力するとAIが企業理念との乖離を分析 |
| 設問別スコア推移 | 同じ設問で練習を重ねると成長が数値で見える（唯一の差別化機能） |
| 設問別履歴・絞り込み | 設問タイトルでフィルタして自分の練習記録を管理できる |
| チュートリアル | 初回起動時に自動表示・右上の「使い方」ボタンでいつでも確認可能 |
| PWA対応 | スマホのホーム画面に追加するとアプリとして使える |

---

## 3. 競合との差別化

| 競合 | 強み | 就跡との違い |
|---|---|---|
| SpeakViz | 音声で話す練習・250問以上・模擬面接 | 就跡は記録が残り設問別の成長が見える |
| リクナビ（ガクチカAI） | ガクチカ素案生成・満足度96% | 就跡は練習を重ねた記録が蓄積される |
| ChatGPT | 何でもできる | 就跡は就活に特化した評価軸・記録が残る |

**唯一の差別化：「設問別のスコア推移で成長が見える」**
→ 他のどのアプリも持っていない機能

## 6. X運用の方針（決定済み）

### アカウントの方向性
- **個人の就活垢**として運用する（アプリ専用垢ではなく）
- 「28卒就活生がアプリを作っている」という文脈を前面に出す
- 就活あるある・開発ログ・拡散投稿の3種類を繰り返す

### 理由
- 就活生のXは「同じ立場の人の話」に反応しやすい文化がある
- 開発者として宣伝するより就活生として共感を取りに行くほうが自然に広がる

### 投稿の3パターン

**① 共感系**
```
面接練習の相手って毎回頼みづらくない？
「また練習付き合って」って言うの申し訳なくて
結局ぶっつけ本番で臨んで後悔するやつ
```

**② 開発ログ系**
```
就活しながらアプリ作ってる
今日やっと志望動機のフィードバック機能が動いた
「御社だからこその理由が薄い」ってAIに言われて
自分のESじゃないのにちょっと刺さった
```

**③ 拡散系**
```
面接・ESをAIに添削してもらえるアプリ作りました
就活中の人、無料で使えるので感想ほしいです
プロフにURL貼ってます
#就活 #27卒 #28卒 #個人開発
```

### プロフィールの方向性
```
名前：名前 / 就活×アプリ開発
bio ：28卒就活生 | 就活ツール作ってます
      面接・ESのフィードバックアプリ → リンク
```
